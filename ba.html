Installation als App

Um dieses Dashboard als eigenständige App auf deinem Tablet zu installieren (mit Offline-Funktion und Vollbild), muss die Datei über einen lokalen Webserver geöffnet werden – nicht direkt als Datei (file://).

Methode 1: Web Server for Chrome (einfach)

1. Installiere die kostenlose App „Web Server for Chrome“ aus dem Google Play Store.
2. Öffne die App, tippe auf „Choose Folder“ und wähle den Ordner, in dem sich diese HTML-Datei befindet.
3. Starte den Server (grüner Knopf).
4. Die App zeigt eine lokale Adresse an, z. B. http://127.0.0.1:8080.
5. Öffne diese Adresse im Browser, tippe unten links auf „App installieren“.

Methode 2: Termux + Python (für Fortgeschrittene)

1. Installiere Termux aus dem Play Store oder F-Droid.
2. Öffne Termux und führe aus: pkg install python
3. Navigiere mit cd /sdcard/Download (oder wohin du die Datei gespeichert hast).
4. Starte den Server: python -m http.server 8080
5. Öffne im Browser: http://127.0.0.1:8080 – dann App installieren.

Nach dem Start des Servers: Schließe dieses Fenster, öffne die lokale Adresse im Browser, und folge dem Installations-Button. Die App funktioniert dann wie eine native App – auch offline!

Hinweis: Ohne Server funktioniert das Dashboard selbstverständlich weiterhin – nur die Installation als App erfordert die obigen Schritte.
